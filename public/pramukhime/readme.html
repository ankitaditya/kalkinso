About PramukhIME JavaScript Library

What is PramukhIME?

PramukhIME is a JavaScript library using which any Web developer can enable the visitor to type in Indian language without downloading/installing any special software.

About Developer

Getting inspiration and blessings from H.D.H. Pramukh Swami Maharaj, Vishal Monpara has dedicated more than 4200 hours from January 2005 till March 2023 to create free utilities for typing in Indian languages.

Supported Browsers

PramukhIME is tested on IE 9+, Chrome, Firefox, Safari and Opera browsers and works as expected.
Even though the script will work in mobile browser Chrome/Firefox, due to lack of standards in Chrome/Firefox mobile browsers, it may not work as expected in rare situation.
